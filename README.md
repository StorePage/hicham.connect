# 🌿 Store Page | Web Design pour Coopératives Marocaines

**Store Page** est un portfolio haut de gamme conçu pour présenter et vendre des solutions web sur mesure destinées aux coopératives et artisans du terroir marocain. Ce projet allie l'authenticité culturelle à une performance digitale moderne.

---

## 🚀 Aperçu du Projet

Ce portfolio est une vitrine interactive permettant de présenter plusieurs modèles de sites web pré-construits. Chaque site est optimisé pour capturer l'âme de l'artisanat marocain tout en offrant des fonctionnalités de vente modernes et une navigation fluide.

## ✨ Caractéristiques principales

*   **🎨 Design Culturel Immersif** : Utilisation des polices *Noto Serif* et *Manrope* avec une palette de couleurs terreuses inspirée du patrimoine marocain.
*   **📱 Système de Prévisualisation Avancé** : Un modal interactif qui simule l'affichage du site sur **Desktop** et **Mobile** en temps réel via des iframes.
*   **📦 Catalogue Diversifié** : Présentation de 9 niches spécifiques (Huile d'olive, Miel, Céramique, Cosmétique bio, Mobilier, etc.).
*   **💬 Conversion Directe** : Intégration fluide de boutons de commande via **WhatsApp** pour une prise de contact et un devis instantanés.
*   **⚡ Performance & SEO** : Développé avec Tailwind CSS pour un chargement ultra-rapide et une structure optimisée.

## 🛠️ Technologies utilisées

*   **Frontend** : HTML5 / CSS3 (Tailwind CSS)
*   **Interactivité** : JavaScript Vanilla (Gestion dynamique du DOM & Modaux)
*   **Typographie** : Google Fonts (Noto Serif & Manrope)
*   **Icônes** : Material Symbols Outlined
*   **Contact** : Intégration API WhatsApp Business

## ⚙️ Installation & Utilisation

Puisque ce projet utilise Tailwind via CDN, aucune étape de compilation complexe n'est requise.

1.  **Cloner le dépôt**
    ```bash
    git clone (https://github.com/StorePage)
    ```
2.  **Lancer le projet**
    Ouvrez simplement le fichier `index.html` dans votre navigateur préféré.
